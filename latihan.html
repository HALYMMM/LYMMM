cara menyimpan gambar pada html dengan code IMG. yang paling mudah adalah menyimpan gambarnya difolder yang sama dengan html
jika gambar ada difolder yang berbeda, maka harus menggunakan URL atau link seperti diawal tadi

dalam upload gambar, ada beberapa kode, yaitu
SRC (gambar yang diambil dari folder atau hyperlink),
ALT (nama dari gambar tersebut jika file gambar rusak),
TITLE (untuk memberikan judul pada gambar),
WIDTH (untuk mengatur ukuran kotak), dan
HEIGHT (tinggi gambar)

namun untuk WIDTH dan HEIGHT sebiaknya hanya menggunakan salah satu, karena jika menggunakan keduany
ditakutkan akan merusak ukuran dari gambar tersebut.
WIDTH dan HEIGHT bisa menggunak ukuran pixel (PX) (100/200/300/400/dst..) atau menggunakan persen (%) (10%/20%/50%/dst....)

untuk menjadikan gambar menjadi hyperlink ialah sebagai berikut
